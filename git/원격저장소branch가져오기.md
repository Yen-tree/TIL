# 원격 저장소의 branch 가져오기

## **원격 저장소 branch 확인**

```python
# 원격 저장소 브랜치 보기
git branch -r

# 로컬 + 원격 브랜치 보기
git branch -a (all)
```

## remote branch 업데이트

```python
git remote update
```

## 원격 저장소의 branch 가져오기

```python
git checkout -t origin/[브랜치명]
```
